NBA Salary Predictor in R

This project focuses on building and comparing two predictive models to estimate NBA player salaries using performance and biometric data. After cleaning and merging datasets containing player stats and salary information from the 2022–2023 NBA season, I explored key variables such as points, rebounds, assists, and net rating. I then trained a Linear Regression model and a Random Forest model to predict salaries and evaluated them using RMSE. The Random Forest model outperformed the linear model in accuracy, particularly for players with lower salaries. This project demonstrates my skills in data wrangling, statistical modeling, and model evaluation using R.
